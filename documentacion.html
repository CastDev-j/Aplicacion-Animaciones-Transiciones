Animaciones y transiciones CSS

Explora y domina la creación de Animaciones y Transiciones dinámicas en tus Páginas Web utilizando CSS.

transition-property

transition-property: background-color;
Especifica las propiedades CSS sobre las cuales se aplicará la transición. Puedes especificar una propiedad específica, como opacity o transform, o usar el valor all para aplicar la transición a todas las propiedades animables.

transition-duration

transition-duration: 0.5s;
Define la duración que tomará realizar la transición de una propiedad, especificada en segundos (s) o milisegundos (ms). Este valor determina qué tan rápido o lento se produce el cambio de estado.

transition-timing-function

transition-timing-function: ease-in-out;
Controla la velocidad de la transición a lo largo de su duración, permitiendo que la transición acelere o desacelere según una curva de velocidad definida. Los valores predefinidos incluyen linear, ease, ease-in, ease-out, ease-in-out, o una función cubic-bezier para una curva de velocidad personalizada. Consulta easings.com para más detalles.

transition-delay

transition-delay: 0.2s;
Establece un tiempo de espera antes de que inicie la transición. Esto puede ser útil para coordinar múltiples transiciones o para esperar que otro evento ocurra antes de comenzar la animación. Al igual que transition-duration, se especifica en segundos (s) o milisegundos (ms).

Atajo de transition

transition: background-color 0.5s ease-in-out 0.2s;
Este atajo de transition permite combinar todas las propiedades individuales de transición (transition-property, transition-duration, transition-timing-function, y transition-delay) en una sola declaración.

@keyframes

@keyframes nombreAnimacion {
from { /* estado inicial */ }
to { /* estado final */ }
}
O utilizando porcentajes para mayor control:
@keyframes nombreAnimacion {
0% { /* estado inicial */ }
50% { /* estado intermedio */ }
100% { /* estado final */ }
}

animation-name

animation-name: miAnimacion;
Especifica el nombre de la animación, el cual debe coincidir con el nombre definido en @keyframes.

animation-duration

animation-duration: 2s;
Define cuánto tiempo toma completar un ciclo de la animación, especificado en segundos (s) o milisegundos (ms).

animation-timing-function

animation-timing-function: ease-in-out;
Describe cómo el ritmo de la animación cambia a lo largo de su duración. Los valores comunes incluyen linear, ease, ease-in, ease-out, y ease-in-out.

animation-delay

animation-delay: 1s;
Establece un tiempo de espera antes de que comience la animación, también en segundos (s) o milisegundos (ms).

animation-iteration-count

animation-iteration-count: infinite;
Indica cuántas veces se debe repetir la animación. Puede ser un número o infinite para una repetición ilimitada.

animation-direction

animation-direction: alternate;
Define si la animación debe alternar la dirección en cada ciclo. Los valores pueden ser normal, reverse, alternate, y alternate-reverse.

animation-fill-mode

animation-fill-mode: both;
Especifica estilos a aplicar antes de que empiece la animación (none, forwards, backwards, both), controlando cómo un elemento animado aparece antes y después de la animación.

animation-play-state

animation-play-state: paused;
Permite pausar y reanudar la animación con paused y running.

Atajo de animation

animation: miAnimacion 2s ease-in-out 1s infinite alternate both;
Puedes combinar todos estos componentes en una sola propiedad animation para simplificar tu código.
